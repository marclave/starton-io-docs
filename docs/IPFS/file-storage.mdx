# Understanding Storage

Welcome to Starton Storage, designed to provide access to IPFS, facilitating the storage and management of digital assets. The service enables users to upload files, folders, and manage NFT metadata through a set of APIs. 

Starton Storage provides comprehensive guides on IPFS, URI, and CID, and offers main API endpoints for various storage operations. Starton Storage ensures content availability by hosting it on dedicated nodes, reducing the need for continuous maintenance and represents a practical solution for users looking to leverage decentralized storage.
