# Pinning CID on IPFS

You can pin content already available on IPFS with Starton using the Content IDentifier (CID).

<Tabs>
<TabItem value="code" label="From Code">

```jsx showLineNumbers
const axios = require("axios");

const startonApi = axios.create({
    baseURL: "https://api.starton.io",
    headers: {
        "x-api-key": "YOUR_API_KEY",
    },
});

startonApi.post("/v3/ipfs/pin", {
    name: "pin name",
    cid: "bafybeid7ybbefggfv6y7gkkbyt3tpi647p66mrmcjkujrlbfbtb6mmbklq",
    metadata: { your: "additionnal", meta: "data" },
})
    .then(res=>console.log(res.data))
    .catch(e=>console.log(e))
```
</TabItem>
<TabItem value="dashboard" label="From Webapp">

1. Go to **Storage**.
1. Click **Upload**.
1. Select **CID**.
1. Enter the **CID** of the content.
1. Enter a **Name** for the file.
1. Click **Upload**.
