# Smart contracts library

**Starton** allows you to access Link tocollection of audited smart contracts** covering most frequent use cases and needs.
It currently includes flavours of the classNameic **ERC20** (cryptocurrency), **ERC721** and **ERC1155\*\* (NFT) smart contracts.

:::info Smart contract Audit
You can find the audit and all of the templates on our **[GitHub repository](https://github.com/starton-io/smart-contract-templates)**
:::
